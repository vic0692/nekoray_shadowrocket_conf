# nekoray_shadowrocket_conf

Генерируются .srs и .list для конфигов nekoray и shadowrocket

### Категории
- ANIME
- CDN
- CHATBOTS
- HENTAI
- IMAGES
- MISC
- PORN
- SOFTWARE
- VIDEO

### Обновление
Релиз собирается автоматически при изменении файлов rules/source/*.txt

### Использование
- nekoray → `rule_set`
- Shadowrocket → `RULE-SET`
